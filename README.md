# learn-plateform

A platform to learn how to be a great frontend developer

## Getting started

You need to have [pnpm](https://pnpm.io/) and [vercel cli](https://vercel.com/docs/cli) installed on your machine.

```bash
pnpm install

vercel env pull

pnpm dev
```
